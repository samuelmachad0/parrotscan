<h1><a id="ParrotScan_0"></a>ParrotScan</h1>
<p>O ParrotScan visa se tornar um software que realiza verificação de plágio através do estudo e aplicação de diversos algoritmos na área.</p>
<p>A ferramenta proposta pretende ser disponibilizada na forma de:</p>
<ul>
<li>API</li>
<li>Plugin integrado ao Moodle</li>
<li>Plataforma Web (App)</li>
</ul>
<p>Nosso maior desejo é combater os diferentes tipos de plágio com um software gratuito e prático de ser utilizado.</p>
